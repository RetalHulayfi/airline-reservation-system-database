# Airline Reservation System Database

## Project Description

This project focuses on designing and implementing a database system for an Airline Reservation System. The system manages information related to passengers, flights, airlines, bookings, and airplanes. The goal is to organize and retrieve airline reservation data efficiently using SQL queries and relational database concepts.

## Technologies Used

* SQL
* MySQL

## Project Features

The database system supports several operations for managing and analyzing airline data, including:

* Retrieving passenger and flight information
* Managing booking details
* Analyzing passenger data using aggregate functions
* Connecting multiple tables using joins
* Performing advanced queries using subqueries

## SQL Operations Implemented

The following SQL techniques were used in this project:

* **Basic SELECT Queries**

  * Passengers from Egypt
  * Flights departing from Riyadh

* **Aggregate Functions**

  * Number of passengers per nationality
  * Average airplane capacity

* **Joins**

  * Booking details with passenger names
  * Flights with airline names

* **Subqueries**

  * Passengers who booked flights to Dubai
  * Flights using specific airplanes

## Tools

* MySQL database system
* SQL queries for data retrieval and analysis

## Project Outcome

The final result of the project was a functional airline reservation database capable of storing and retrieving flight, passenger, and booking information efficiently.
